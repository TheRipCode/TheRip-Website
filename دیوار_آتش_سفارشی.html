صفحه اصلی پروژه‌ها

توضیح کلی پروژه

پروژه‌ای برای کنترل ترافیک ورودی و خروجی شبکه با استفاده از قوانین سفارشی.

پیش‌نیازها

  • آشنایی با IP و پورت‌ها
  • استفاده از iptables در لینوکس

کدهای کامل

# Example iptables rule to block an IP sudo iptables -A INPUT -s 192.168.1.100 -j DROP # Example rule to allow specific port sudo iptables -A INPUT -p tcp --dport 22 -j ACCEPT # Save the rules sudo iptables-save > /etc/iptables/rules.v4

توضیح کدها

این دستورات iptables ترافیک ورودی از IP خاص را مسدود و پورت SSH را مجاز می‌کند.

آموزش اجرا

دستورات را با دسترسی sudo در ترمینال لینوکس اجرا کنید.
