توضیح کلی پروژه

ابزاری برای جمع‌آوری اطلاعات از منابع مختلف به منظور تحلیل و شناسایی.

پیش‌نیازها

  • آشنایی با پایتون
  • کتابخانه requests و BeautifulSoup

کدهای کامل

import requests from bs4 import BeautifulSoup def gather_info(url): response = requests.get(url) soup = BeautifulSoup(response.text, 'html.parser') print("Title of the page:", soup.title.string) url = "http://example.com" gather_info(url)

توضیح کدها

این اسکریپت اطلاعات متای یک وب‌سایت، مانند عنوان صفحه، را استخراج می‌کند.
