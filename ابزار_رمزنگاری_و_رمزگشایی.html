کدهای کامل

from cryptography.fernet import Fernet # Generate and save a key key = Fernet.generate_key() with open('key.key', 'wb') as key_file: key_file.write(key) # Encrypt data cipher = Fernet(key) encrypted = cipher.encrypt(b"Sensitive Data") print("Encrypted:", encrypted) # Decrypt data decrypted = cipher.decrypt(encrypted) print("Decrypted:", decrypted.decode())

توضیح کدها

این کد از کتابخانه cryptography برای رمزنگاری و رمزگشایی متن استفاده می‌کند.

آموزش اجرا

کد را اجرا کنید و فایل 'key.key' تولید شده را برای رمزگشایی ذخیره کنید.
